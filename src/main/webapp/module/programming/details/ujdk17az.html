Ubuntu JDK1.7安装

测试时间：2015年6月 测试环境：Linux Ubuntu 14.04.1 64位

1、官网下载 JDK 安装包
这一步没什么好说的，请大家务必到官网去下载，养成个好习惯

最新的 JDK 都已经出了 1.8 了

2、解压文件并移动到目标路径
使用命令 sudo /usr/local/jdk 在 local/ 目录下新建名为 jdk 的文件夹
找到下载的 JDK 的安装包，使用命令 gzip -d jdk-7u75-linux-x64.tar.gz 解压 gzip 压缩形式
解压完成后，会生成 jdk-7u75-linux-x64.tar 文件
使用命令 tar -xvf jdk-7u75-linux-x64.tar 解包出现一个名为 jdk1.7.0_75 的文件夹
使用命令 sudo mv jdk1.7.0_75 /usr/local/jdk/ 将文件夹移动到新建的 jdk/ 目录下

用户自己安装的东西放到 /usr/local 下是个好习惯

3、增加环境变量
使用命令 sudo vi /etc/profile 编辑 profile 文件
行至文件末尾 增加如下内容：
# java_home 路径
export JAVA_HOME=/usr/local/jdk/jdk1.7.0_75
# jre_home 路径
export JRE_HOME=${JAVA_HOME}/jre
# path
export PATH=$PATH:${JAVA_HOME}/bin
# classpath
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib

有人将这些内容增加到 ~/.bashrc 中，个人比较倾向于放到 /etc/profile 中

Linux 严格区分大小写

4、配置默认的 JDK 版本
使用命令 sudo update-alternatives --install /usr/bin/java java /usr/local/jdk/jdk1.7.0_75/bin/java 300
使用命令 sudo update-alternatives --install /usr/bin/javac javac /usr/local/jdk/jdk1.7.0_75/bin/javac 300
然后执行命令 sudo update-alternatives --config java

若系统中没有 JDK，则会提示 here is only one alternative in link group java (providing /usr/bin/java): /usr/local/jdk/jdk1.7.0_75/bin/java

若系统中原有 JDK 如：openjdk 等，则需要输入数字选择刚安装的版本

5、确认安装成功
使用命令 java -version 如果出现下图信息，则表明安装成功

出现这个画面，说明您已经安装成功了，写个经典的"hello world！"测试一下吧
