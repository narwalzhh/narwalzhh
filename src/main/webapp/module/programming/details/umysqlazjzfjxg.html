Ubuntu Mysql数据库安装及字符集修改

测试时间：2015年6月 测试环境：Linux Ubuntu 14.04.1 64位

1、安装 Mysql 数据库
首先使用命令 sudo apt-get update 更新源
更新完成后使用命令 sudo apt-get install mysql-server mysql-client 安装 Mysql 数据库

如果直接执行后面这条命令，可能会无法下载，这时候就要更新一下源了

2、安装过程中
安装过程中会有选择和输入 root 用户密码，选择使用的是 tab 键，而密码自己输入一个自己满意的就可以了

3、登录 Mysql 数据库
使用命令 mysql -u root -p 并“回车”，输入自己设置的密码进入 Mysql 数据库

-p后面可以直接跟密码的

4、修改字符集编码为 utf-8 并取消大小写敏感
使用命令 sudo vi /etc/mysql/my.conf，修改配置文件，需添加如下配置：
找到 [client] 添加
default-character-set=utf8
找到 [mysqld] 添加
character-set-server=utf8
init_connect='SET NAMES utf8'
lower_case_table_names=1

和 utf8 相关的时修改字符集，剩下的那一条是取消数据库对大小写敏感的

5、重启 Mysql 数据库验证字符集是否修改完成
使用命令 sudo service mysql restart 重启数据库，进入数据库
执行命令 show variables like "%char%"; 如观察到下图，证明已修改成功

至此，您已成功设置了 Mysql 数据库，赶快体验一下吧
