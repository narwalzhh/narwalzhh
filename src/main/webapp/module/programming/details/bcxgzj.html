一、Elipse 设置 Alt + / 代码提示功能

测试时间：2015.07 测试环境：ubuntu eclipse kepler


进入 Eclipse，选择 Window >> preferences >> General >> Keys ，搜索 word completion ，在右半部分窗口中出现下图界面：



起初 Word Completion 中绑定的是 Alt + / ，解除绑定 Unbind Command 然后搜索 Content Assist ，给它增加绑定 Alt + / ，保存后，在编辑区试一下就会出现想要的效果了！



在 windows 下的 eclipse 好像不用这么配置就可以使用 Alt + /，但是在 linux 版本下就是不行，已经配置了两次了，怕再给忘了，就记录一下吧
